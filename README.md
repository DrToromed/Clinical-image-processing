# Clinical-image-processing
A Python-based image processing tool for chest X-ray enhancement  and annotation, built with PIL/Pillow.
## Features
- Grayscale conversion
- Auto-contrast enhancement
- Histogram equalization
- Sharpness filter
- Side-by-side comparative view
- Anatomical annotation with ImageDraw

## Technologies
- Python 3
- PIL/Pillow

## Use Case
Designed for clinical settings to improve visualization of 
radiological findings such as pulmonary infiltrates and 
cardiac abnormalities.
